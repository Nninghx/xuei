## 项目概述

本项目是方便UP我本身提高效率的小工具，需要有一点点的开发知识即可，提前部署好开发环境，就可以本地离线运行，由于是Alpha版，可能会出现bug! 本up非专业开发者，这些小工具经过简单测试，可以正常使用

Tu_Pian_He_Cheng-Alpha.py 可将多张图片按不同布局模式合成一张图片。支持JPG/PNG/BMP/GIF等常见图片格式的输入和输出。

Tu Pian Zhuan ico-Alpha.py 可以把图片转成ico格式文件
